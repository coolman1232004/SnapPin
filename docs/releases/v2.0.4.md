# SnapAnchor 2.0.4

SnapAnchor 2.0.4 refreshes the compact toolbar artwork for **Text** and **Pin**.

## Improved

- Text now uses a simple upright **T** without the previous bottom serif.
- Pin now uses a rounded, tilted outline thumbtack based on the supplied visual reference.
- Both icons retain SnapAnchor's shared 18 px size, rounded 1.7 px stroke, hover state, and active teal state.

## Validation

- Full x64 build and localization audit pass with zero warnings or errors.
- The complete smoke suite passes.
- The rendered compact-toolbar preview confirms both icons remain clear and aligned at 100% scaling.
