## SnapPin 1.2.11

SnapPin 1.2.11 makes the default capture toolbar closely follow Snipaste's compact workflow while keeping SnapPin's additional tools available when needed.

### New default toolbar

- Annotation tools: **Rectangle, Arrow, Pencil, Marker, Blur, Text, Eraser**.
- Undo and Redo remain permanently positioned and become available when there is annotation history.
- Capture actions: **Cancel, Pin, Save, Copy**.
- Ellipse, straight line, magnifier, thumbnail pin, quick save, scrolling capture, recording, OCR, and recapture remain available from Toolbar Preferences.

### Preferences layout

- Annotation toolbar settings now appear on the left.
- Capture toolbar settings now appear on the right.
- The **Default** buttons restore the new Snipaste-style selections and ordering.
- Existing untouched defaults migrate automatically; user-customized layouts are preserved.

### Downloads

- **Installer:** `SnapPin-Setup-win-x64.exe`
- **Portable:** `SnapPin-Portable-win-x64.zip`

### Requirements and security

- Windows 10 version 1809 or later, x64.
- This build is not code-signed and may trigger Windows SmartScreen.
- SHA-256 values are included in `release.json` for verification.
