## SnapPin 1.2.5

SnapPin 1.2.5 completes the interface-language work. Selecting Simplified Chinese or Traditional Chinese now changes the complete application after restart—not only the Preferences tabs.

### What changed

- Localized dashboard, Preferences, capture overlay, annotation tools, pinned-image menus, OCR, recording, history, update dialogs, and notification-area menu
- Added application-wide automatic localization for WPF windows, controls, context menus, tooltips, and table headers
- Added a localized **Restart now** / **Later** prompt after changing the interface language
- Added a safe restart handoff that waits for the previous SnapPin process before reopening
- Fixed Traditional Chinese conversion so window titles never receive an extra trailing character

### Validation

- Complete static interface-label coverage audit
- Simplified and Traditional Chinese regression tests
- Visual verification of the Chinese dashboard, compact Preferences layout, and restart dialog
- Release build and full x64 smoke-test suite
