# SnapAnchor 2.1.1

SnapAnchor 2.1.1 fixes pinned-image annotation input and makes capture and pin tool selection behave consistently.

## Fixes

- Pinned-image drawing tools now receive pointer drags over the image instead of the transparent overlay passing them through.
- The pinned-image toolbar is positioned from the pin's actual on-screen bounds and stays directly beneath it.
- Clicking the active annotation tool again returns to move mode on a pin.
- Clicking the active annotation tool again unselects it during normal screen capture as well.

## Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

The installer and portable build are currently unsigned, so Windows SmartScreen may display a warning. SnapAnchor requires 64-bit Windows 10 version 1809 or later, or Windows 11.
