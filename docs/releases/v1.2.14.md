## SnapAnchor 1.2.14

SnapAnchor 1.2.14 fixes the missing or generic icon shown on Windows taskbar buttons.

### Taskbar icon fix

- The dashboard now explicitly uses the high-resolution SnapAnchor logo as its window icon.
- Preferences, History, custom capture, and the portable updater use the same icon.
- Explorer, notification-area, title-bar, and taskbar branding are now consistent.
- Automated icon checks and an isolated live Windows taskbar review verify the result.

### Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

### Requirements and security

- Windows 10 version 1809 or later, x64.
- This build is not code-signed and may trigger Windows SmartScreen.
- SHA-256 values are included in `release.json` for verification.
