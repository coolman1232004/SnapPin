# SnapAnchor 2.1.4

SnapAnchor 2.1.4 keeps pinned captures sharp, correctly sized, and in their original position on mixed-DPI dual-monitor computers.

## Fixes

- A capture pinned on a 125%-scaled monitor now opens at its native physical pixel size and reports **100%**, not 80%.
- Initial pin bounds are reapplied after Windows completes the destination monitor's DPI transition, preventing position and size drift.
- The selection outline is now drawn over the image without consuming image space or rescaling the screenshot.
- Pinned images use pixel-preserving rendering, avoiding the slight blur previously visible on both notebook and external displays.

## Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

The installer and portable build are currently unsigned, so Windows SmartScreen may display a warning. SnapAnchor requires 64-bit Windows 10 version 1809 or later, or Windows 11.
