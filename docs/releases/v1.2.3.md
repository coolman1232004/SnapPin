## SnapPin 1.2.3

This update improves capture resizing and completes the GitHub update workflow.

### Fixed

- The capture toolbar hides while an edge or corner resize handle is being dragged.
- When resizing finishes, the toolbar returns automatically and anchors below the adjusted capture region.
- In annotation mode, the properties row hides and returns together with the main toolbar.

### GitHub updates

- SnapPin uses the official GitHub `release.json` feed by default.
- Automatic startup update checks remain optional in Preferences.
- **Check for updates…** is now available directly from the system-tray menu.
- Downloaded installers are verified against the release SHA-256 hash before launch.
- Existing settings with an empty update URL migrate automatically to the official feed.

### Downloads

- `SnapPin-Setup-win-x64.exe` — guided Windows installer
- `SnapPin-Portable-win-x64.zip` — self-contained portable build
- `release.json` — update metadata and SHA-256 hashes

### Requirements

- Windows 10 version 1809 (build 17763) or later
- x64 processor

This release is not code-signed. Windows SmartScreen may display an unknown-publisher warning.
