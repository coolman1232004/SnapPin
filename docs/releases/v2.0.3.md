# SnapAnchor 2.0.3

SnapAnchor 2.0.3 fixes the pinned-image **Show toolbar** workflow.

## Fixed

- The original pinned image remains in its exact screen position while the toolbar is open.
- No replacement or duplicate capture surface appears beside the toolbar.
- The capture-style annotation toolbar is anchored directly below the pin and kept on-screen.
- A visible **Cancel** action closes the toolbar and leaves the original pin unchanged.

## Validation

- Full x64 build and localization audit pass with zero warnings or errors.
- The complete smoke suite verifies the transparent edit layer, Cancel action, stable toolbar anchor, and lossless flattening.
