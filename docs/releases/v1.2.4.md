## SnapPin 1.2.4

This release adds safe, automatic updates for portable SnapPin copies and simplifies the GitHub update interface.

### Portable self-updates

- Portable copies automatically select the portable ZIP instead of launching the installer.
- Downloads are verified against the SHA-256 hash published with the GitHub release.
- New files are staged before SnapPin closes.
- Existing program files are backed up before replacement.
- SnapPin restarts automatically when the update completes.
- If replacement fails, backed-up files are restored and the previous copy is reopened.

### Cleaner update settings

- **Check for updates when SnapPin starts** remains available.
- The internal JSON manifest address is no longer displayed or editable.
- Preferences now links to the public [SnapPin GitHub page](https://github.com/coolman1232004/SnapPin).
- Installed copies continue to use the guided Windows installer.

### Downloads

- `SnapPin-Setup-win-x64.exe` — guided Windows installer
- `SnapPin-Portable-win-x64.zip` — self-contained portable build with in-place updates
- `release.json` — update metadata and SHA-256 hashes

### Requirements

- Windows 10 version 1809 (build 17763) or later
- x64 processor

This release is not code-signed. Windows SmartScreen may display an unknown-publisher warning.
