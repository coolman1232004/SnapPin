## SnapPin 1.2.6

SnapPin 1.2.6 focuses on maintainability, update clarity, and real-world Windows compatibility.

### Highlights

- See download percentage and status while SnapPin updates, with safe cancellation before installation begins.
- Complete rare-message localization backed by standard English, Simplified Chinese, and Traditional Chinese resource files.
- More reliable sizing on mixed-DPI, portrait, tiny, and negative-coordinate monitor layouts.
- Stale audio-device selections automatically fall back to the Windows default device.
- Diagnostics now identify local or remote sessions and describe the connected display layout.
- Large capture, annotation, and pinned-image classes are split into focused components without changing their behavior.

### Validation

- 481 localized resource keys and 425 visible UI strings audited.
- Update progress, cancellation, mixed DPI, portrait displays, remote sessions, stale devices, and 12-frame long captures covered by automated tests.
- Full x64 release build and smoke suite pass with zero warnings or errors.
