## SnapAnchor 1.2.15

SnapAnchor 1.2.15 adds a safe way to cancel selectable OCR text on pinned images.

### Text-selection cancellation

- A **Cancel** action now appears beside **Copy** and **All text** after selecting recognized text.
- Cancel clears the highlight and closes the selection toolbar.
- Pressing Escape clears the selection instead of closing the pinned image.
- Clicking empty image space also clears the selection.
- OCR text-selectable mode remains enabled, so you can immediately select different text.

### Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

### Requirements and security

- Windows 10 version 1809 or later, x64.
- This build is not code-signed and may trigger Windows SmartScreen.
- SHA-256 values are included in `release.json` for verification.
