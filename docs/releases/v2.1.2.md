# SnapAnchor 2.1.2

SnapAnchor 2.1.2 fixes pinned-image toolbar placement across mixed-DPI, multi-monitor desktops and unifies the update workflow with the main application design.

## Fixes

- The pinned-image toolbar now stays attached below the pin when the pin is on a secondary monitor.
- Physical monitor coordinates are converted using the annotation overlay's DPI, avoiding notebook-to-monitor scaling drift.
- Moving a pin while its toolbar is open remains synchronized across different display scales.
- Update available, download progress, update ready, and portable updater windows now use the same warm off-white, charcoal, and warm-grey palette as the dashboard.

## Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

The installer and portable build are currently unsigned, so Windows SmartScreen may display a warning. SnapAnchor requires 64-bit Windows 10 version 1809 or later, or Windows 11.
