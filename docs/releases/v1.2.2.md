## SnapPin 1.2.2

The first public SnapPin release brings the complete local-first capture workflow to Windows.

### Highlights

- Region, window, UI-element, full-screen, delayed, fixed-size, and scrolling capture
- Inline annotation with editable shapes, arrows, drawing, text, blur, eraser, and undo/redo
- Always-on-top image pins with groups, OCR-selectable text, zoom, opacity, and session restore
- Offline OCR in English, Simplified Chinese, Traditional Chinese, and Japanese
- QR/barcode recognition and selectable text inside pinned images
- MP4/GIF recording with pause/resume, review, trim, keep, or discard
- Searchable local history with favourites, tags, repeat capture, and recycle bin
- Configurable hotkeys, toolbars, output formats, and privacy exclusions
- Verified GitHub update feed with SHA-256 installer validation

### Downloads

- `SnapPin-Setup-win-x64.exe` — guided Windows installer
- `SnapPin-Portable-win-x64.zip` — self-contained portable build
- `release.json` — version, download URL, sizes, and SHA-256 hashes

### Requirements

- Windows 10 version 1809 (build 17763) or later
- x64 processor

### Security note

This release is not code-signed. Windows SmartScreen may show an unknown-publisher warning. Download only from this repository's Releases page and compare the file hash with `release.json`.
