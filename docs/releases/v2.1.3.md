# SnapAnchor 2.1.3

SnapAnchor 2.1.3 provides a stronger fix for pinned-image toolbar placement on mixed-DPI, dual-monitor computers.

## Fixes

- The pinned-image annotation overlay now covers only the monitor containing the pin instead of spanning monitors with different display scales.
- Toolbar coordinates and available space come from the destination monitor's native bounds and DPI.
- When a pin is moved to another display while editing, the overlay is rehomed and realigned to that display.
- The update workflow retains the warm-neutral and charcoal design introduced in 2.1.2.

## Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

The installer and portable build are currently unsigned, so Windows SmartScreen may display a warning. SnapAnchor requires 64-bit Windows 10 version 1809 or later, or Windows 11.
