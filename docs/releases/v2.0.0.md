# SnapAnchor 2.0.0

SnapAnchor 2.0.0 is the first release under the new product name. It also rebuilds the capture and annotation toolbar as a compact, consistent WPF vector interface.

## Product rename

- The application, executable, installer, portable package, namespaces, assets, local storage, update feed, and documentation now use **SnapAnchor**.
- Existing settings, history, pinned sessions, diagnostics, output defaults, and startup registration migrate automatically on first launch.
- Installed upgrades remove obsolete executable, shortcut, registry, and application files only after the new payload has been copied successfully.
- Existing portable 1.x copies can update in place through a hidden one-release compatibility launcher, which is not retained by SnapAnchor after the transition.
- The exact pre-rename source is preserved in the `pre-snapanchor-v1.2.15` tag.

## Compact vector toolbar

- Replaced mixed inline toolbar artwork with a shared library of sharp WPF vector paths.
- Compact mode uses 18 px icons in 28 px buttons on a 34 px light surface.
- Added consistent slate ink, teal active states, subtle hover states, 40% disabled opacity, rounded line caps and joins, restrained separators, and a light shadow.
- Capture and annotation overlays share the same spacing, geometry, active state, and DPI-aware layout metrics.
- Preserved configurable tool visibility, order, properties, and all existing capture actions.

## Validation

- Release x64 build completes with zero warnings or errors.
- The complete smoke suite passes, including localization alignment, capture resizing, toolbar stability, annotation rendering, OCR, history, recording, updater, installer, and 100%, 125%, and 150% DPI layout checks.
