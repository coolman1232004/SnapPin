## SnapAnchor 1.2.12

SnapAnchor 1.2.12 makes the Preferences window smaller and prevents settings from becoming inaccessible on scaled or low-resolution displays.

### Compact Preferences

- The dialog is now 620 x 470 logical pixels by default.
- Tabs, fields, buttons, headings, margins, and the toolbar configuration lists use tighter sizing.
- Every Preferences tab scrolls vertically when its content is taller than the available area.
- The two-column Toolbar tab also supports horizontal scrolling if the window is narrowed further.
- **Cancel** and **Save changes** remain fixed and visible at the bottom while page content scrolls.

### Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

### Requirements and security

- Windows 10 version 1809 or later, x64.
- This build is not code-signed and may trigger Windows SmartScreen.
- SHA-256 values are included in `release.json` for verification.
