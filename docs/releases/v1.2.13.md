## SnapAnchor 1.2.13

SnapAnchor 1.2.13 adds a separate daily GitHub update schedule.

### Daily update checks

- Enable **Check for updates every day** in Preferences > About.
- SnapAnchor checks at startup when no check has been attempted on the current local calendar day.
- If SnapAnchor remains open across midnight, it checks within 30 minutes.
- A manual **Check now** action counts as that day's check.
- **Check for updates when SnapAnchor starts** remains an independent option and still checks on every launch when enabled.
- The new option is disabled by default.

### Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

### Requirements and security

- Windows 10 version 1809 or later, x64.
- This build is not code-signed and may trigger Windows SmartScreen.
- SHA-256 values are included in `release.json` for verification.
