# SnapAnchor 2.0.5

SnapAnchor 2.0.5 makes the pinned-image toolbar behave like capture mode while preserving direct pin movement.

## Improved

- Opening **Show toolbar** starts in move mode instead of preselecting a drawing tool.
- Dragging the image moves the original pin and keeps the toolbar attached.
- Clicking an active drawing tool again deselects it and returns to move mode.
- The toolbar uses capture mode's exact measured-width placement calculation.
- Drawing, editing, Copy, Save, Finish, and Cancel continue to operate on the pinned image.

## Validation

- Full x64 build and localization audit pass with zero warnings or errors.
- Regression tests verify capture-identical placement, move-first state, tool toggle-off behavior, pin movement, and lossless image flattening.
