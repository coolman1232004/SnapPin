# SnapAnchor 2.0.2

SnapAnchor 2.0.2 completes the portable cleanup introduced in 2.0.1.

## Changes

- Fresh portable ZIPs contain only `SnapAnchor.exe`.
- In-place updates now explicitly back up and remove the obsolete previous-name executable even when it was not listed in the earlier package manifest.
- SnapAnchor also performs a safe startup cleanup for existing extracted folders that were updated or overlaid manually.
- Pinned-image OCR uses an I-beam over recognized selectable words and the normal arrow over empty image areas.

## Validation

- Complete smoke suite passes with zero build warnings or errors.
- Portable update cleanup is covered for unmanaged previous-name apphosts.
- ZIP and extracted-folder inspection confirm zero legacy-name files.
