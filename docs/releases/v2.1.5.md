# SnapAnchor 2.1.5

SnapAnchor 2.1.5 is a follow-up mixed-DPI fix for external-monitor pin size, sharpness, and first-open toolbar placement.

## Fixes

- Pins now hold their exact native screen rectangle while Windows completes the destination monitor's DPI transition.
- The screenshot fills that rectangle exactly instead of appearing smaller than the selected monitor area.
- Window shadow rendering no longer rasterizes or softens the screenshot layer.
- The annotation toolbar uses the overlay window's native monitor DPI before its first visible layout, preventing the initial jump to the monitor's lower-left corner.
- Moving or resizing a pin immediately hands control to the user and stops initial placement stabilization.

## Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

The installer and portable build are currently unsigned, so Windows SmartScreen may display a warning. SnapAnchor requires 64-bit Windows 10 version 1809 or later, or Windows 11.
