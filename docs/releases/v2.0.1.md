# SnapAnchor 2.0.1

This maintenance release cleans up the portable package and improves selectable OCR text in pinned images.

## Changes

- Removed the obsolete previous-name compatibility executable from the portable ZIP and extracted application folder.
- The pinned-image pointer changes to an I-beam only while it is over a recognized selectable word.
- Empty image areas use the normal arrow pointer, and leaving recognized text restores the arrow immediately.
- Text dragging keeps the I-beam active until selection finishes.

## Validation

- Release x64 build and complete smoke suite pass with zero warnings or errors.
- Portable ZIP inspection confirms that it contains only the SnapAnchor application host.
- OCR cursor hit-testing is covered for recognized text, empty image space, and active selection.
